API для учёта сотрудников компании. Разработка BackEnd идёт на Java, FrontEnd разрабатывается на Flutter

Авторизация (прибытие и отход) на месте работает через скан случайного QR-кода.

Из кода удалить ненужный функционал (Кеширование, статистика). Упростить структуру приложения. Ориентир - разработка именно мобильного приложения.
